# LIVE-NEWS-APP
Responsive application which shows realtime news ( live ) feed for various categories using Redux.

## Functionalities
  * various categories 
    - headlines
    - buisness
    - entertainement
    - sports
    - health
    - science
    - general
    - technology
  * display feed in cart layout which includes
    - image
    - heading 
    - author name,
    - createTime,
    - link to feed.
